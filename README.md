# 心理學量表預試 - Silicon Sampling Skill

一個專為**心理學量化研究預試階段**設計的 Claude Agent Skill，透過生成具有完整成長經驗背景的細緻 persona，模擬目標母群在自陳式量表上的填答反應。

## 核心特色

- **專注預試階段**：樣本數設計為 30-60 人，符合量表編製預試的需求規模
- **細緻 Persona**：每個 persona 都具備完整的成長經驗敘事（150-300 字）
- **三層填答推理**：依據成長經驗 → 人格特質 → 情境因素進行填答模擬
- **NEO-PI-R 30 構面**：使用完整的 30 構面人格模型
- **心理學真實性**：確保人格-經驗-填答之間的心理動力連貫性

## 適用情境

此技能專為以下情境設計：

| 情境 | 說明 |
|------|------|
| 量表編製預試 | 新編量表在正式施測前的預先測試 |
| 量表修訂預試 | 修訂既有量表後的初步驗證 |
| 跨文化適應預試 | 翻譯或文化適應量表的預先測試 |
| 教學研究示範 | 研究方法課程中的量表發展教學 |

## 快速開始

### 範例 1：量表編製預試

```
User: 我正在編製一份「學業拖延量表」，想要進行預試。
      目標母群是台灣大學生，需要 40 個樣本。

Agent: 好的，我將協助您進行學業拖延量表的預試模擬。
       請提供您的量表題目和量尺說明...
```

### 範例 2：使用預設母群

```
User: 請幫我用 PHQ-9 進行預試，30 個樣本，使用預設的大學生母群。

Agent: 好的，我將使用「台灣大學生」母群進行 PHQ-9 的預試模擬...
```

## 細緻 Persona 結構

每個 persona 都包含：

### 基本資料
- 年齡、性別、教育程度、職業、居住狀況

### NEO-PI-R 30 構面人格剖面
- 神經質 (N1-N6)：焦慮、憤怒敵意、憂鬱、自我意識、衝動、脆弱
- 外向性 (E1-E6)：熱情、合群、獨斷、活力、尋求刺激、正向情緒
- 開放性 (O1-O6)：想像、審美、感受、嘗試、理念、價值
- 親和性 (A1-A6)：信任、坦誠、利他、順從、謙虛、溫柔
- 盡責性 (C1-C6)：勝任感、條理、責任、追求成就、自律、深思熟慮

### 成長經驗敘事（150-300 字）
- 原生家庭背景與教養風格
- 重要的成長經驗或轉折事件
- 人際關係模式的形成
- 自我概念與價值觀的發展
- 當前生活情境與主要壓力源

### 填答傾向摘要
- 基於人格特質和成長經驗的填答預測

## 樣本數建議

| 樣本數 | 適用情境 | 可進行的分析 |
|--------|----------|--------------|
| 30 人 | 最小預試樣本 | 題目難度、基本信度估計 |
| 40 人 | 一般預試 | 項目分析、刪題決策 |
| 50 人 | 標準預試 | 探索性因素分析（謹慎解讀） |
| 60 人 | 完整預試 | 較穩定的因素結構探索 |

## 輸出格式

### 1. responses.csv - 填答資料

```csv
respondent_id,ITEM1,ITEM2,ITEM3,...
R001,4,2,5,...
R002,3,4,3,...
```

### 2. personas.csv - Persona 資料 + 30 構面

```csv
respondent_id,age,gender,education,...,N1_anxiety,N2_hostility,...,C6_deliberation
R001,21,female,bachelor_3,...,58,35,...,55
```

### 3. persona_narratives.md - 成長經驗敘事

每個 persona 的完整成長經驗敘事，以 Markdown 格式呈現。

### 4. summary.yaml - 摘要統計

包含人口統計摘要、人格特質分布、填答統計、預試品質建議。

## 重要原則

### 🚫 絕對不可使用腳本

- 不可使用程式碼生成 persona
- 不可使用隨機數生成器填答
- 每個 persona 和填答都必須經過深思熟慮的推理

### ✅ 確保心理學真實性

- 成長經驗必須能合理解釋人格特質
- 人格特質必須一致地影響填答
- 相關題目的填答必須呈現合理相關

## 研究倫理說明

使用此技能時，請注意：

1. **透明度**：在研究報告中明確說明使用了 AI 模擬數據
2. **效度限制**：AI 模擬的反應可能無法完全代表真實人類
3. **用途限制**：僅適用於預試階段，不應取代正式施測
4. **補充性質**：作為量表發展過程的輔助工具
